# リスト内包表記とは

## ① 何をしたかったか

for文で書いていた「条件に合う要素だけ集めてリストを作る」処理を、
もっと短く書きたかった。

```python
# 元のコード
result = []
for x in range(10):
    if x % 2 == 0:
        result.append(x)
```

## ② 何が起きたか / 気づいたこと

リスト内包表記という書き方を使うと、同じ処理が1行で書けることを知った。

```python
result = [x for x in range(10) if x % 2 == 0]
```

## ③ どう理解したか

「新しいリストを作る」ときの for + if は、1行にまとめられる。
ただし条件が複雑になると読みにくくなるので、そのときは
普通のfor文に戻す方が良さそう。

読みやすさ・短さのトレードオフとして覚えておく。
